Тестовое задание 1 - https://webbylab.com/

Необходимо реализовать веб-приложение для хранения информации о фильмах. Вместе с реализацией требуется предоставить документ, объясняющий архитектуру приложения и содержащий инструкцию по запуску приложения.

Информация о фильме:
1. Уникальный идентификатор
2. Название 
3. Год выпуска
4. Формат (VHS, DVD, Blu-Ray)
5. Список актеров (“Имя и фамилия актера”)

Функции, которые должна поддерживать система:
1. Добавить фильм
2. Удалить фильм
3. Показать информацию о фильме
4. Показать список фильмов, отсортированных по названию в алфавитном порядке   
5. Найти фильм по названию.
6. Найти фильм по имени актера.
7. Импорт фильмов с текстового файла (файл прилагается). Файл должен загружаться через веб-интерфейс.

В первую очередь, важно, чтобы была реализована вся функциональность с точки зрения пользователя.
Задание необходимо реализовать на PHP. Данные должны храниться в базе данных MySQL. Без использования фреймворков. Проект должен запускаться на PHP local server. В описании проекта должна быть детальная информация о том как запускать проект: как развернуть Б.Д., файл конфигурации и т.п.

================================================================
Архитектура приложения:
Приложение построено с использоанием модели MVC. Бекенд - PHP, фронтенд - HTML, CSS, Bootstrap 3, JavaScript, база данных - MySQl.

Установка и запуск приложения:
1) Скопировать папку с приложением на сервер. Желательные характеристки сервера:
- Apache 2.4
- PHP 5.6
- MySQL 5.5
2) Открыть файл index.php
3) Указать в нем параметры подключения к базе данных: константы HOST, USER, PASSWORD, DATABASE
4) Установить значение константы DATABASE_INSTALLED в false
5) Открыть стартовую страницу приложения. Будет автоматически создана база данных и таблицы
6) Установить значение константы DATABASE_INSTALLED в true. База данных и таблицы уже созданы - нет необходимости повторно вызывать функцию инсталляции
7) Открыть страницу http://moviestorage/upload и импортировать данные о фильмах из файла sample_movies.txt. Можно создать новый файл для импорта фильмов - но структура в нем должна повторять структуру данных в sample_movies.txt. Сам файл можно увидеть под формой импорта.
